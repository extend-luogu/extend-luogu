## 添加到 cph / CPH import

### 分类

普通模块

### 功能

在题目页面内将题目推送至 [cph](https://github.com/agrawal-d/cph) 中

### 设置

- 自动隐藏按钮 / Auto Hide Button
  - 当无法连接服务时隐藏按钮。
- 自定义题目名称格式（非比赛）/ Custom Problemname Template (Not Contest)
  - 自定义题目页面传输名称
- 自定义题目名称格式（非比赛）/ Custom Problemname Template (Contest)
  - 自定义比赛页面传输名称
  - ${cid} 表示比赛编号
  - ${pid} 表示题目编号（带首字母）
