## 用户首页 HTML 显示 / User Intro Ins

### 分类

普通模块

### 功能

解析用户主页的 HTML 代码。
有 XSS 保护。

用法：

在个人主页写下：

`exlg.html: <HTML 代码>`

`exlg.html-paste: <luogu paste id>`

`[](exlg.xxx:xxx)`（中间不得有空格）

即可解析。
